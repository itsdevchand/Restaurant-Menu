# 🍽️ Elegant Restaurant Menu Page

A responsive and beautifully structured restaurant menu template built for a premium dining establishment. This project showcases how to elegantly combine **CSS Grid** (for layout management) and **CSS Flexbox** (for alignment constraints) to create an adaptive fluid interface.

## ✨ Features

- **Hybrid Layout Engine:** Uses CSS Grid for section columns and automatic responsive wrapping, alongside CSS Flexbox for item/price alignment.
- **Anchor Navigation:** Smooth-scrolling anchor links for quick, seamless page jumping.
- **Responsive Architecture:** Auto-fit item grids automatically convert from a multi-column desktop showcase to a clean, readable single-column view on mobile panels.
- **Editorial Typography System:** Uses a sophisticated mix of serif and sans-serif fonts to mimic high-end printed menus.
